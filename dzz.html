У веб-розробці часто використовується термін HTML - це мова розмітки для створення веб-сторінок. DOM є програмним представленням структури сторінки, що дозволяє змінювати її через JavaScript.
Також важливим є CSS для стилізації контенту.

При розробці інтерфейсів важливо розуміти концепцію UX, яка охоплює досвід користувача при взаємодії із системою.
API дозволяє програмам взаємодіяти між собою. Також існує REST - стиль архітектури для веб-сервісів.

Сучасна розробка часто включає JS - потужну мову програмування для браузерів. CMS - це система керування контентом, наприклад WordPress.
Для верстки важливий Flexbox, що дозволяє створювати гнучкі макети.

Терміни веб-розробки

HTML

Мова розмітки для створення веб-сторінок.

HTML (HyperText Markup Language) - це стандартна мова розмітки для створення веб-сторінок. Вона визначає структуру контенту за допомогою системи тегів та атрибутів.

Основні сучасні стандарти: HTML5, XHTML. Використовується разом з CSS та JavaScript для створення повноцінних веб-додатків.

Дізнатися більше
DOM

Програмне представлення структури сторінки.

DOM (Document Object Model) - це програмний інтерфейс для веб-документів. Він представляє сторінку таким чином, що програми можуть змінювати структуру, стиль і вміст документа.

DOM представляє документ у вигляді дерева вузлів, де кожен вузол представляє частину документа (елемент, текст тощо).

Дізнатися більше
CSS

Мова стилів для оформлення веб-сторінок.

CSS (Cascading Style Sheets) - це мова таблиць стилів, яка використовується для опису зовнішнього вигляду документа, написаного HTML.

Сучасні стандарти включають CSS3 з такими можливостями як анімації, переходи, медіа-запити для адаптивного дизайну, змінні (CSS Custom Properties) тощо.

Дізнатися більше
UX

Досвід користувача при взаємодії із системою.

UX (User Experience) охоплює всі аспекти взаємодії кінцевого користувача з компанією, її сервісами та продуктами.

Основними принципами хорошого UX є: зручність використання, доступність, ефективність, емоційна взаємодія та задоволення від використання продукту.

Дізнатися більше
API

Інтерфейс для взаємодії програм між собою.

API (Application Programming Interface) - це набір визначень та протоколів для створення програмного забезпечення.

У веб-розробці, API часто є набором HTTP-запитів для отримання або відправлення даних (REST, GraphQL, SOAP). Вони дозволяють різним системам спілкуватися між собою.

Дізнатися більше
REST

Стиль архітектури для веб-сервісів.

REST (Representational State Transfer) - це архітектурний стиль для розподілених систем, зокрема для веб-сервісів.

Основні принципи REST: клієнт-серверна архітектура, відсутність стану (stateless), кешування, уніфікований інтерфейс, шарова система. Часто використовується з протоколом HTTP.

Дізнатися більше
JS

Мова програмування для веб-додатків.

JavaScript - це високорівнева, інтерпретована мова програмування, яка є однією з основних технологій Всесвітнього павутиння.

Сучасний JavaScript (ES6+) включає такі можливості як класи, проміси, async/await, модулі. Використовується як на клієнтській (браузер), так і на серверній (Node.js) стороні.

Дізнатися більше
CMS

Система керування контентом.

CMS (Content Management System) - це програмне забезпечення для створення, редагування та керування цифровим контентом.

Популярні CMS: WordPress (блоги, сайти), Drupal (складні сайти), Joomla, Shopify (електронна комерція). CMS дозволяють керувати контентом без глибоких технічних знань.

Дізнатися більше
Flexbox

Модель для гнучкої верстки.

Flexbox (Flexible Box Layout Module) - це CSS3-модуль для розміщення, вирівнювання та розподілу простору між елементами у контейнері, навіть коли їх розмір невідомий або динамічний.

Ключові концепції: flex-контейнер, flex-елементи, основна та поперечна осі, вирівнювання, порядок відображення елементів.
